Thank you so much for downloading my script

How to use it :
LDtk( file's name, tile set in GMS, optional : the number for the first tile)
exemple : LDtk(file, tkTile, 93)
		   LDtk("test.ldtk", tkTest)

What is "the number for the first tile" :
-> GMS delete the first tile for an empty one.
    If you use this tile number 0 then you might have moved it,
    so now you can give the new number of this tile 0.
